 HirePath 🚀

HirePath is a full-stack internship and job portal designed to connect students with employers through a streamlined application process.

 🛠 Tech Stack
Frontend: React, Tailwind CSS, React Router, Axios
Backend: Node.js, Express, Helmet, express-validator, express-rate-limit
Database: MongoDB
Authentication: JWT

 ✨ Features
Student Dashboard: Monitor application status and track progress.
Employer Portal: Manage job listings and provide applicant feedback.
Mobile Responsive: Fully optimized UI for mobile devices and small screens.

 🚀 How to Run Locally

1. Backend Setup
1.  Navigate to the server directory.
2.  Install dependencies: npm install.
3.  Create a .env file with MONGO_URI and JWT_SECRET.
4.  Start the server: npm start.

2. Frontend Setup
1.  Navigate to the client directory.
2.  Install dependencies: npm install.
3.  Create a .env file with VITE_API_URL.
4.  Start the development server: npm run dev.
